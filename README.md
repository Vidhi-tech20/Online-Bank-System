# Online Bank Mini System

This project is a Python Flask web application for an online banking mini system, designed for public deployment. It features secure transaction validations, clean modular code, error handling, and a simple UI. Ready for deployment to Render, Vercel, or any public cloud platform.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Transfer Money (with KYC and balance validation)
- Account listing
- Error handling and validation
## UI Screens
- Account creation
- Transaction (deposit, withdraw, transfer)
- Account listing
- Output display panel

pip install -r requirements.txt
python app.py
## Getting Started

1. Install dependencies:
	```bash
	pip install -r requirements.txt
	```
2. Run the app:
	```bash
	python app.py
	```

## Deployment
Deploy to Render, Vercel, or any public cloud platform. Add deployment steps in `.github/workflow.yml` or follow platform-specific instructions.
